# Score de Crédito com Machine Learning

Projeto de Machine Learning desenvolvido em Python para prever o score de crédito de clientes com base em dados históricos.

O modelo é treinado para identificar padrões nos dados e realizar previsões que podem auxiliar em processos de análise de crédito.

## Funcionalidades

* Tratamento e preparação de dados
* Treinamento de modelo preditivo
* Avaliação de performance do modelo
* Previsão de score de crédito para novos clientes

## Tecnologias Utilizadas

* Python
* Pandas
* Scikit-learn
* Jupyter Notebook

## Etapas do Projeto

1. Importação dos dados
2. Limpeza e preparação dos dados
3. Treinamento do modelo de Machine Learning
4. Avaliação do modelo
5. Realização de previsões com novos dados

## Estrutura dos Dados

O projeto utiliza dois arquivos principais:

* **clientes.csv** → base de dados utilizada para treinar o modelo
* **novos_clientes.csv** → dados utilizados para realizar novas previsões

## Como Executar

1. Clonar o repositório

```shell
git clone https://github.com/viniciusdevrr/machine-learning-score-credito.git
```

2. Instalar as dependências

```shell
pip install pandas scikit-learn
```

3. Abrir o projeto no VS Code

Abra a pasta do projeto no VS Code e localize o arquivo:

```
modelo_score_credito.ipynb
```

4. Executar o notebook

Abra o arquivo **modelo_score_credito.ipynb** e execute as células do notebook para treinar o modelo e gerar as previsões.

## Objetivo do Projeto

Aplicar técnicas de Machine Learning para resolver problemas de previsão utilizando dados históricos de clientes.
